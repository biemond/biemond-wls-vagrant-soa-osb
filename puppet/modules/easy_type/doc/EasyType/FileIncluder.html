- (Object) include_file(name)

This is an implementation of a C like include. You can use this at places where Puppet doesn't support parent-classed like when defining Custum Types. include_file uses the already established search path. If you need to include a file from a subdircetory, use the subdirectory name

Examples:


include_file 'puppet/types/ora'

Parameters:

  • name (String) —

    this is the name of the file to be included

Returns:

  • the evaluated content of the file

Raises:

  • (ArgumentError) —

    when the specified file is not found

def include_file(name)
  full_name = get_ruby_file(name)
  raise ArgumentError, "file #{name} not found" unless full_name
  eval(IO.read(full_name), nil, full_name)
end
